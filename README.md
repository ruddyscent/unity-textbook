# The Unity Textbook (7th Edition) – Updated for Unity 6

This repository contains the source code from **"The Unity Textbook (7th Edition)"**,  published by **Gilbut Publishing Co., Ltd.**, migrated to work with **Unity 6**.

The following section includes the original README provided by the publisher.

## 《유니티 교과서, 개정 7판》

《유니티 교과서, 개정 6판》의 개정판입니다.<br/>
Unity 2023 버전에 맞춰 업데이트했습니다 (macOS, Windows).<br/>

## 예제 파일에 대하여

이 파일에는 **주식회사 SB 크리에이티브**에서 발간한  
《유니티 교과서 2023 완벽 대응》에 수록된 리소스 파일과 스크립트 데이터가 포함되어 있습니다.<br/>

책의 내용과 각 샘플, 그리고 샘플 내에서 사용하는 데이터는 모두 **저작권법의 보호**를 받습니다.<br/>
책 내용 및 샘플이나 데이터의 전부 또는 일부를 무단으로 복사, 복제·전재하는 것은 법으로 금지되어 있습니다.<br/>

각 샘플의 저작권은 저작자에게 있으며, 프로젝트 및 프로그램의 개변(수정·응용)은 허용됩니다.<br/>
단, 게재된 샘플을 허가 없이 복사, 복제·전재하는 것은 금지됩니다.<br/>

이 파일 내의 샘플은 사용자의 책임하에 사용해야 하며,  
사용 중 발생하는 문제에 대하여 저작자 및 출판사(주식회사 SB 크리에이티브)는 일체 책임을 지지 않습니다.

## 컴퓨터 바이러스에 대하여

만일을 대비하여 사용자는 반드시 **바이러스 검사를 수행**하고 안전성을 확인해야 합니다.<br/>
이미 바이러스에 감염된 컴퓨터에 본 파일을 복사한 경우, 복사된 파일도 감염될 수 있습니다.<br/>
이에 대해 제작자는 어떠한 책임도 지지 않습니다.
